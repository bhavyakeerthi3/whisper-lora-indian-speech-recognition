# Results

This directory is reserved for experiment outputs generated from real speech datasets.

Do not replace `TBD` values with estimated numbers. Only publish metrics produced by `src/indic_whisper_asr/evaluation/evaluate_asr.py` or an equivalent reproducible evaluation script.

Recommended files:

- `baseline_whisper_metrics.json`
- `lora_metrics.json`
- `qlora_metrics.json`
- `language_breakdown.csv`
- `training_resource_usage.csv`
